  程序使用Docker部署后，日期不正确

### 方法 A：在 docker run 时添加环境变量（推荐）
```
docker run -e TZ=Asia/Shanghai ...
```

### 方法 B：在 Dockerfile 中设置
```
ENV TZ=Asia/Shanghai
RUN ln -snf /usr/share/zoneinfo/$TZ /etc/localtime && echo $TZ > /etc/timezone
```
### 方法 C：在 docker-compose.yml 中设置
```
services:
  your-api:
    environment:
      - TZ=Asia/Shanghai
```
